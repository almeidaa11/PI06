Sobre a WikiHero

Um projeto heróico

Nosso projeto é voltado para o público que curte heróis e personagens.

Trata-se de um projeto feito em HTML, CSS, JavaScript e Bootstrap.

Nossa equipe se empenhou ao máximo para trazer um resultado incrível com uma experiência única.

Veja outros diferenciais:

  • Utilização de ferramentas modernas
  • Layout responsivo para todos os dispositivos
  • Ideia única
  • Contato direto entre cliente-desenvolvedor
  • Desenvolvimento com metodologia ágil
